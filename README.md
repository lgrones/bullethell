# bullethell

A bullet-hell / danmaku game built in [Godot](https://godotengine.org/) with C#.

## ⚠️ Not a commercial product

This project is **not** a commercial product and is **not** for sale. It exists
purely as a personal project for **learning game development** — bullet pattern
systems, rendering pipelines, gameplay architecture, and the tooling around
them. It is shared for educational reference only.
